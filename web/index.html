How It Works

Upload

POST /api/upload with your token image. Get a hosted URL.

curl -X POST https://api.bags.fm/upload
-H 'x-api-key: YOUR_KEY'
-F 'image=@token.png'

Launch

POST /api/token/launch with name, symbol, and image. Token goes live in seconds.

curl -X POST https://api.bags.fm/token/launch
